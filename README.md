# OVO VIP 联机百家乐 v7

本版本基于 v6 继续优化，保留 Firebase 房间同步：倒计时、开奖、补牌、路形图、所有玩家下注金额同步。

## v7 更新

- 洗牌逻辑改为双条件：
  - 同一牌靴达到 70 局后，下一局开始前自动洗牌；
  - 或牌靴剩余不足 80 张时，下一局开始前自动洗牌。
- 兼容旧房间数据：如果旧房间没有 `shoeRoundNo` 字段，会用已有历史局数迁移判断，70 局以上会在下一局触发洗牌。
- 路图统计改成彩色球 + 数字：
  - 蓝球 + 数字 = 闲赢次数
  - 红球 + 数字 = 庄赢次数
  - 绿球 + 数字 = 和局次数
  - 红球内 6 + 数字 = 幸运6次数
  - 蓝球内 7 + 数字 = 幸运7次数
- 继续保持八副牌牌靴、先发四张暗牌、倒计时结束后一秒翻一张、翻完前四张后再判断是否补牌。
