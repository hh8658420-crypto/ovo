## 使用

把本包里的 `index.html`、`firebase-config.js`、`firebase-rules-demo.json` 覆盖到你的 GitHub 仓库根目录。GitHub Pages 访问时可以继续用：

`https://你的域名/?room=VIP001&v=新版时间戳`

同一个 `room` 参数进入同一房间，不同设备会同步倒计时、下注、开牌结果和路单。
