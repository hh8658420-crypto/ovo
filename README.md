# OVO VIP 联机百家乐 v6

本包基于当前联机版继续修改，保留 Firebase 房间同步、倒计时同步、开奖同步、路形图同步、所有玩家下注金额同步。

v6 修改重点：

- 下注 15 秒倒计时期间只预发 4 张暗牌：闲①、庄①、闲②、庄②。
- 倒计时结束后才进入开牌，每 1 秒翻 1 张。
- 翻完前四张后，严格按百家乐补牌规则判断是否补第三张，不再提前显示第三张暗牌。
- 补牌从同一牌靴继续抽牌，按闲补牌、庄补牌顺序显示。
- 使用 8 副牌牌靴，共 416 张牌。
- 牌靴状态写入 Firebase 房间状态，所有玩家使用同一牌靴进程。
- 洗牌点写入代码：牌靴剩余不足 52 张时，下一局开始前自动重洗 8 副牌。
- 下方开奖层放大，覆盖取消/重复/筹码区域。
- 下方开奖层无标题、无外框，闲蓝色、庄红色，随上方每翻一张同步更新点数。
- 路形图统计增加：闲赢多少把、庄赢多少把、和局多少把、幸运6多少把、幸运7多少把。

上传方式：

把本包中的 index.html、firebase-config.js、firebase-rules-demo.json 覆盖到仓库根目录。

建议访问时加缓存参数，例如：

?room=VIP001&v=20260703-v6
