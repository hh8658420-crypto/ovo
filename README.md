# OVO VIP 联机百家乐 v9

本版基于 v8-sync 修复手机端下方实时开奖点数显示问题。

## v9 修改

- 修复手机端下方实时开奖层点数被牌面挤掉的问题。
- 新增 `max-width:430px` 手机适配，覆盖 iPhone 393px/414px 等常见宽度。
- 下方开奖层保持：左闲蓝、右庄红、暗牌/翻牌同步、点数实时计算。
- 小屏手机自动缩小下方牌面，优先保证 `闲/庄点数` 可见。

## 保留功能

- Firebase 联机同步
- 15 秒下注倒计时
- 倒计时期间预发前四张暗牌
- 倒计时结束后每秒翻一张
- 严格百家乐补牌规则
- 8 副牌牌靴与洗牌机制
- 路形图与统计球
- 所有玩家下注同步
