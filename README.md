# yyy 版界面修复包

本包只基于用户上传的 yyy.zip 修改。

改动：
- 玩法区按副玩法 / 正玩法分组，文字居中对齐。
- 闲、庄主玩法等宽等高。
- 去掉圆形灵动岛风格计时器，改为扁平倒计时。
- 倒计时固定 15 秒，时间到自动发牌开牌，不需要手动按钮。
- 发牌、开牌改为逐张显示，并且每张牌显示后都会更新闲/庄点数。
- 补牌会显示在牌框内。
- 开奖完成后进入下一轮自动清空牌面。
- 筹码下注增加飞筹码动画。
- 筹码金额为 100 / 500 / 1000 / 5000 / 10K。
- 百分比圆标调小并提高透明度，按全场正副玩法总下注计算。
- 本金显示在筹码下方区域。
