# 原界面联机同步版

这是基于你上传的源码修改的版本，杂音频文件没有处理，界面保持原版。

## 已改内容

- 同步倒计时
- 同步开奖结果
- 同步开奖记录/路单
- 同步所有玩家下注数字
- 本机下注仍只用于本机本金结算，别人下注不会影响你的本金
- 不生成别人的筹码动画

## 上传方法

1. 解压本 ZIP。
2. 把里面的全部文件上传到 GitHub 仓库根目录。
3. 覆盖旧文件并 Commit。
4. 打开：`https://hh8658420-crypto.github.io/ovo/?v=yuan-sync-1&room=VIP001`

如果 Firebase 规则没有开放测试写入，可以参考 `firebase-rules-demo.json`。
